# Raspberry-Pi
Scrips for the Raspberry Pi

**Pi Updater:** It fully updates your Raspberry Pi's software, firmware, and cleans up leftovers, then optionally reboots.
Run in terminal:
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20Updater.sh | bash
```

**Pi System Info:** It displays a quick snapshot of your Pi's CPU, RAM, disk, temperature, network, and OS info in one clean readout.
Run in terminal:
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20System%20Info | bash
```
