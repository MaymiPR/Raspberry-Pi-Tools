# Raspberry-Pi
Tools for the Raspberry Pi

**Pi Updater:** It fully updates your Raspberry Pi's software, firmware, and cleans up leftovers, then optionally reboots.
Run in terminal:
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20Updater.sh | bash
```

**Pi System Info:** It displays a quick snapshot of your Pi's CPU, RAM, disk, temperature, network, and OS info in one clean readout.
Run in terminal:
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20System%20Info.sh | bash
```

**Pi Cleaner:** It cleans up unused packages, cache, old logs, and temp files to free up disk space.
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20Cleaner.sh | bash
```

**Pi Docker Status:** It shows all running and stopped Docker containers along with installed images.
```bash
curl -s https://raw.githubusercontent.com/MaymiPR/Raspberry-Pi/refs/heads/main/Pi%20Docker%20Status.sh | bash
```
